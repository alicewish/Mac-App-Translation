An application doesn't open anymore

If an application doesn't open anymore or seems blocked on a message displaying "Caching Fonts", you have probably removed the fonts cache.

â– 

In this case, don't force the application (including X11) to quit; wait for the fonts cache to rebuild.

Indeed, the rebuilding of fonts caches in some open source applications (GIMP, MPlayer OSX Extended, and so on) takes a particularly long time.

After the cache is rebuilt, the application will open again normally.
